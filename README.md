# Project Goals
An interactive Tableau dashboard project analyzing FIFA World Cup data.
This project provides insights into tournament statistics, team performance, player achievements, and historical trends.

# Project Overview
The FIFA World Cup Tableau Project aims to:
Visualize historical World Cup statistics
Compare team and player performance across tournaments
Analyze goals, matches, and win/loss trends
Identify patterns in tournament results and performance metrics

# Features
Tournament Insights
Total matches, goals, and average goals per match
Tournament winners and runner-ups
Team Performance Analysis
Win/loss/draw records per team
Goals scored and conceded by each team
Historical performance trends
Player Statistics
Top scorers across tournaments
Player awards (Golden Boot, Best Player)
Goal distribution by match and tournament
Match Analysis
Goals per match, penalty outcomes, and scorelines
Match outcomes by stage (group, knockout, final)
Interactive Filters
Filter by tournament year, team, player, or stage
Drill-down to individual match details

# Insights
Top scoring teams and players per tournament
Historical goal trends and match outcomes
Comparison of group stage vs knockout stage results
Performance of host nations
Trends in tournament winners and runners-up
Visualizations

Line charts, bar charts, heatmaps, and pie charts

Geospatial maps for host countries
